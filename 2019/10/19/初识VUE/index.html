1.1 VUE简述

Vue.js 是一套响应式系统，前端开发库。Vue自问世以来所受关注度不断提高，是非常流行的前端开发框架。

1.2 Vue的主要特点

1.轻量级的框架

2.双向数据绑定

3.指令

4.组件化

5.客户端路由

6.状态管理

1.3 Vue的安装

#查看版本

按下 win + r 键 输入cmd

输入指令 npm -v

#由于npm安装速度慢，因此可以使用淘宝镜像及其命令cnpm

cnpm install vue

#全局安装 vue-cli

cnpm install –global vue-cli

​

创建一个基于webpack模板的新项目，项目名称为project

vue init webpack project

cd project

cnmp install //安装项目依赖

cnpm run dev //运行项目

成功执行以上命令后访问 ： localhost：8080 即可。
